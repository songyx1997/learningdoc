#### 1. HTML

##### 1.1基本结构

乱码：编码与解码所采用的字符集不同。

```html
<!-- 文档声明用于告诉浏览器当前网页的版本 -->
<!DOCTYPE html>
<!-- 规定网页的语言 -->
<html lang="zh">
    <head>
        <!-- 告知浏览器网页的字符集，避免乱码 -->
        <meta charset="utf-8">
        <!-- 搜索引擎通过title判断网页主要内容 -->
        <title>网页的标题</title>
    </head>
    <!-- 网页的主体 -->
    <body></body>
</html>
```
